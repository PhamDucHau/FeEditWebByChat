# Nike Shoe Website - Trang Web Bán Giày Nike

Trang web bán giày Nike được xây dựng bằng React với giao diện hiện đại và đẹp mắt.

## Cấu Trúc

- **Header**: Navigation bar với logo Nike, menu điều hướng và các nút tìm kiếm/giỏ hàng
- **4 Sections**:
  1. **Hero Section**: Section giới thiệu chính với slogan "Just Do It"
  2. **Featured Products**: Hiển thị các sản phẩm nổi bật
  3. **Features Section**: Giới thiệu các ưu điểm của cửa hàng
  4. **Newsletter Section**: Form đăng ký nhận tin
- **Footer**: Thông tin liên hệ, liên kết và mạng xã hội

## Cài Đặt

```bash
npm install
```

## Chạy Ứng Dụng

```bash
npm start
```

Ứng dụng sẽ chạy tại [http://localhost:3000](http://localhost:3000)

## Công Nghệ Sử Dụng

- React 18
- CSS3 với responsive design
- React Scripts
